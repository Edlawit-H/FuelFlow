# ⛽ FuelFlow: Real-Time Fuel Queue Management System

FuelFlow is a smart digital solution designed to eliminate fuel station congestion. It allows drivers to join queues remotely, track their position in real-time, and enables station managers to regulate fuel flow efficiently.

## 🏗️ Project Architecture

The project is divided into two main parts:
- **/frontend**: A modern React SPA (Single Page Application) built with Vite and Tailwind CSS.
- **/backend**: A Node.js/Express REST API using MongoDB for real-time data persistence.



## 🎨 Frontend Features (Current Progress)
Developed with a focus on UI/UX and professional branding:
- **Driver Dashboard**: Real-time station discovery with fuel availability status.
- **Queue Management**: "Join Queue" logic with predicted position and estimated wait times.
- **Admin Portal**: Station-side controls for managing active pumps and fuel types.
- **Responsive Design**: Built using Tailwind CSS for a high-end mobile-first experience.

## ⚙️ Backend Features
- **Queue Logic**: Real-time Estimated Wait Time (EWT) calculation.
- **Authentication**: Secure JWT-based user and admin authentication.
- **GeoJSON Integration**: Geospatial indexing for finding nearby stations.
- **RBAC**: Role-Based Access Control (Driver vs. Station Admin).
